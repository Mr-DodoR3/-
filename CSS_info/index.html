CSS

CSSはこんなWebページをデザインするためのスタイルシート言語です。

文法

CSSはHTMLで記入された文字に様々な装飾をするための言語です。

HTMLで記入したものを装飾するための言語のため、事前にHTMLでCSSを読み込むプログラムを記入する必要があります。

↑「HelloWorld」と表示させるプログラム

プロパティ

color: カラーコード; 文字の色指定を行います。
background-color: カラーコード; 背景の色指定を行います。
font-size: サイズpx; フォントサイズを指定します。
font-family: フォント名; フォントを指定します、たしかフォント名に空白があればダブルクォーテーションで囲む必要あり。
width: 横幅px; 横幅を指定します、画像サイズの指定とかによく使う。
height: 縦幅px; 縦幅を指定します、画像サイズの指定とかによく使う。
float: 並び; 要素を横並びにする、LEFTとRIGHTを指定できる。
padding: 幅px; 要素の外側の全ての方向に幅px分の空白を開ける。
padding: 上下幅px 左右幅px; 要素の外側に幅px分の空白を開ける、引数の一つ目が上下、引数の二つ目が左右。
padding: 上幅px 右幅px 下幅px 左幅px; 要素の外側に幅px分の空白を開ける、引数の一つ目が上、引数の二つ目が右、引数の三つ目が下、引数の四つ目が左。
padding-方向: 幅px; 要素の外側に幅px分の空白を開ける、方向は"top"(上),"right"(右),"bottom"(下),"left"(左)が指定可能。
margin: 幅px; 要素の内側の全ての方向に幅px分の空白を開ける。
margin: 上下幅px 左右幅px; 要素の内側に幅px分の空白を開ける、引数の一つ目が上下、引数の二つ目が左右。
margin: 上幅px 右幅px 下幅px 左幅px; 要素の内側に幅px分の空白を開ける、引数の一つ目が上、引数の二つ目が右、引数の三つ目が下、引数の四つ目が左。
margin-方向: 幅px; 要素の内側に幅px分の空白を開ける、方向は"top"(上),"right"(右),"bottom"(下),"left"(左)が指定可能。
list-style: none; HTMLのリストの・や数字を消します。
border: 太さpx 種類 カラーコード 要素の周りに線を描画します。
一つ目の引数が太さ、二つ目の引数に種類、三つ目の引数にカラーコードを記入する。
線の種類は以下のものが存在する
  1. solid == 普通の実線
  2. double == 二重線
  3. groove == くぼんでいるような線
  4. ridge == 盛り上がって見えるような線
  5. inset == 全体がくぼんでいるような線
  6. dotted ==: 点線
  7. dashed == 破線
text-decoration: 種類 文字に下線や波線などの装飾を付けます。
線の種類は以下のものが存在する
  1. underline == 下線
  2. line-through == 取り消し線
  3. wavy == 波線
  4. double == 二重線
  5. dotted == 点線
  6. dashed ==: 破線
