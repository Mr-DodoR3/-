Java

メジャーでいろんなことができるプログラミング言語です。

具体的にはWebアプリケーションやパソコンのソフトウェア、スマホのアプリ開発に使われます。

ちなみにこの部活（というかこの学校）で最もよく使われる言語です。知らんけど。

文法

Hello World

「Hello World」と表示するプログラムです。

index.java

3行目の"System.out.println("文字列")は文字を表示させるメソッドである。

Javaでは分の終わりには";"(セミコロン)を付ける必要がある。(制御文とかは除く)

1行目と2行目は...Javaを動かすおまじないだ！(適当)

4行目と5行目の"}"は1行目と2行目を"{}"で囲む必要があるからある。

変数・配列

変数と配列の宣言についてのプログラムです。

VariableTemp.java

変数の宣言は [型 変数名 = 初期値;] です。

"="は数学と違って代入を表すので注意してください。


また、変数の型は以下の通りです。（主に自分がよく使うもの抜粋）

int 16bitの整数型
long 32bitの整数型
float 単精度浮動小数点型（ただの32bitの小数点型だと思ってもらってOK）
double 倍精度浮動小数点数型（ただの64bitの小数点型だと思ってもらってOK）
char 文字型、1文字を表す型で文字は''で囲んで表す。
String 文字列型、charとは違い文章を表す型で、文字列は""で囲んで表す。
boolean 真偽の2値を扱う型、trueかfalseしか入らない。

制御文

条件分岐文

物事を判断し、行動を変えるための制御文です。

BranchTemp.java

Javaの条件分岐は主にif文とswitch文があります。

8~16行目がif文、19~29行目がswitch文です。

ちなみに、elseif文を一つにまとめた三項演算子(32行目)というものも存在します。

繰り返し文

同じ処理を何度も繰り返すための処理です。

LoopTemp.java

Javaの繰り返し文は主にfor文、do文、while文があります。

for文は指定数回繰り返すときに、while文は条件が満たしてるときに、do文はwhile文と同様条件を満たしているときに繰り返されるが、最低でも1度はプログラムが実行されるのが特徴。

ちなみに先輩によるとdo分やwhile文よりfor分のほうがいいらしいです。

クラス

Javaはオブジェクト指向の言語のため、クラスを作ることが可能です。

Hello.java

今回はあくまでJavaの文法の書き方なので詳しいクラスの説明はしませんが、クラスとは設計図のようなものです。

例えば、シューティングゲームを作るとき、弾が複数同時に発射したいとなったとき、クラスを使えばこの設計図をもとに弾を作ってくれます。(作ったものはインスタンスという)

3行目はコンストラクタといい、インスタンスが作られたタイミングで呼び出されるメソッドです、こちらはクラス名と同名の必要があります。

8行目はクラスのメソッドです。

"public"はアクセス修飾子というもので、このメソッドはどこから扱えるかを示しています。

"public"は外部のインスタンスからでもメソッドが扱えることを示します。

関数一覧(関数以外のやつもかなりあるが...)

double = tan(double ラジアン値); 引数のラジアン値の正接(タンジェント)を返します。
