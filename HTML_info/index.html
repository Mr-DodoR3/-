HTML

HTMLはこんなWebページの開発に使われる文字を表示するための言語です。

文法

HTMLは普通のプログラミング言語と違ってif文とかfor文とかはありません。

表示させたい文や画像をタグと呼ばれるもので囲むことで文字を表示させることができます。

↑「HelloWorld」と表示させるプログラム

タグ一覧

<head>初期設定記入</head> ページの設定等を記入します。
<title>文章</title> ページのタイトルを設定します。（head内に記入）
<meta charset="文字コード"> 使用する文字コードを設定します、といってもだいたいUTF-8でいいですが...（head内に記入）
<link rel="icon" href="URL"> ウェブページのアイコンを読み込みます。（head内に記入）
<link rel="stylesheet" href="読み込むCSSファイル.css"> CSSファイルを読み込みます。（head内に記入）
<body>本文</body> 表示する本文を記入する場所です。これ以降は全てbody内に記入するタグです。
<div>子要素</div> レイアウト指定の要素であり、画面上に表示させるものの親要素としてよく使われます。
<h1>見出し</h1> 見出しを記入します、ちなみに見出しはh1からh6までの種類が存在します。
<p>文字列</p> 段落を表す要素です、説明等の本文によく用いられます。
<a href="URL">文字列</a> hrefで指定したURLに飛びます。
<img scr="URL" alt="説明"> scrで指定した画像を表示します、altは表示できない場合の代替テキストの設定です。
<ul>リスト要素</ul> 後術するli要素の親要素で、先頭に黒点を付けます。
<ol>リスト要素</ol> 後術するli要素の親要素で、先頭に連番を付けます。
<li>文字列</li> 箇条書きにして文章を表示します。
<span>文字列</span> 主に文の一部で用いるものであり、文の一部分にのみ別のCSSを適応させる。
<br> 改行させます。
<script src="URL"></script> srcで指定したJavaScriptを読み込みます。
<input> 1行のテキスト入力を受け付ける要素です。
<input type="submit" value="値"> 送信ボタンの要素を意味する要素で、type="submit"は送信ボタンであることを示し、valueでボタンのテキストをその値に変更します。
<textarea></textarea> 複数行のテキスト入力を受け付ける要素です。。
